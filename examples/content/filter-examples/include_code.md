# Code Includer

The `include_code` filter provides a function that can make a code sample from a file, pulling in just specific lines if desired.

Call it like so:

```
{{ '{{' }} include_code("examples/sample_include.json", lines="1,3,5-6", mark_disjoint=True) {{ '}}' }}
```
